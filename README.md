# API Developer Portal — Seguros Bolívar

Portal de desarrolladores para integración de APIs de seguros en el ecosistema colombiano de Open Insurance.

## Módulos

- **Onboarding & Seguridad** — Registro de aliados, validación de identidad, términos Open Insurance, credenciales sandbox
- **Catálogo & Documentación** — Catálogo interactivo de APIs por ramos, documentación Swagger, manifiesto AI
- **AI Playground** — Consola de pruebas con lenguaje natural para agentes de IA

## Tech Stack

- Next.js 14 (App Router), TypeScript, Tailwind CSS, Lucide React
- OAuth 2.0 + OIDC, JWT en httpOnly cookies, RBAC
- OpenTelemetry, Correlation-ID, Circuit Breaker, Rate Limiter
- Vitest + fast-check (Property-Based Testing)
